# ai_snap_detect

Smart safety monitoring projects (PPE detection, face recognition, sign detection, voice transcription, and scenario search).

## Model Weights

Large model files should be downloaded from external storage instead of being committed to Git.

See [MODEL_DOWNLOADS.md](MODEL_DOWNLOADS.md) for:
- recommended hosting locations,
- required file paths,
- download commands,
- and verification checks.

## Expected Local Paths

After downloading, these files should exist:
- `ai_projects_hub/models/yolo9c.pt`
- `ai_projects_hub/models/yolo9e.pt`

Face embeddings file:
- `ai_projects_hub/face_db/embeddings.json`
